# **Chatty**
An application that allows users to text and message with others.  Users can create a username and chatroom, and chat with others
within the rooms they've created!

## **Technologies Used**

### Front-End
* HTML
* CSS
* Javascript

### Back-End
* NodeJS
* Express
* Socket io
